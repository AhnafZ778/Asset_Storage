### 1. Extraction Strategy

for the **Summer** theme:

1.  **CSS**: Keep `[THEME 5] SUMMER`.
2.  **Animations**: Keep `[ANIMATION] SUMMER`.
3.  **Cleanup**: Delete other theme blocks.

### 2. HTML Setup

- Set `<body>` class to `theme-summer`.
- Include `<!-- [THEME SEGMENT] SUMMER DECORATIONS -->`.

### 3. Key Features

- **Sun Pulse**: The SVG sun in the corner has a gentle pulse animation.
- **Butterflies**: CSS-based flight paths using `@keyframes flutter`.

### 4. Standalone Card Usage (Component Only)

To use just the UI cards:

1.  **CSS**:
    - Copy `body.theme-summer` (variables).
    - Copy the `/* CARD STYLING */` section.
2.  **HTML**:
    - Extract `.cards-container` and the modal HTML.
    - Apply `theme-summer` class to your wrapper div.
3.  **JS**:
    - Copy the card click listener and modal logic.
